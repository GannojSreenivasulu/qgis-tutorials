บทนำ¶

QGIS คือ ซอฟต์แวร์โอเพนซอร์ส สำหรับงานด้าน GIS คุณภาพสูง ที่ได้รับความนิมอย่างแพร่หลาย เอกสารนี้จะประกอบไปด้วยชุดของวิธีการใช้งานและกลเม็ดในการใช้งานเพื่อแก้ไขปัญหาต่างทางด้าน GIS

คุณสามารถใช้เอกสารนี้ในการเรียนรู้การใช้ซอฟต์แวร์นี้ได้ด้วยตนเอง อย่างไรก็ตาม หากคุณค้นเคยกับ QGIS สักหน่อยแล้ว คุณสามารถเลือกหัวข้อที่สนใจได้ทันที โดยที่ไม่ต้องเรียนตามลำดับในเอกสารนี้แต่อย่างใด

คุณสามารถเข้าไปดูที่ QGIS Learning Resources ในนี้จะมีสื่อการสอนแหล่งอื่นๆ ที่ช่วยให้คุณเรียนรู้ QGIS ได้เช่นกัน

การร่วมพัฒนา¶

เอกสารนี้ถูกจัดเตรียมโดยใช้ reStructuredText ซึ่งสามารถเข้าถึงซอร์สโค้ดได้ที่ github
